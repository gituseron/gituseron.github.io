Medis-ONE

의료용 디스플레이 생산 공정 통합 관리 시스템 (MES)

Project Overview

의료용 디스플레이 제조 공정의 고도화된 품질 관리를 위해 **자재 입고부터 생산, 품질 검사, 최종 출하 및 사후 추적**까지 전 과정을 디지털화하고 통합 관리하는 MES 솔루션입니다. 특히 의료기기 품질경영시스템(ISO 13485) 기준을 참고하여 **데이터의 무결성**을 확보하는 데 집중했습니다.

🛠 Tech Stack

Spring Boot JPA / Hibernate MySQL React REST API

01. 공정 분석 및 시스템 모델링

* 본 단계는 실제 제조 도메인을 분석하여 시스템 데이터로 모델링하는 과정에 집중했습니다.

STEP 01. 입고 및 표면 처리: 원자재 입고 확인 및 Unique Lot ID(바코드) 부여
STEP 02. 광학 본딩 (핵심): 레진 도포량 및 경화 시간 임계값(기준치) 모니터링 관리
STEP 03. 제품 조립: 작업지시서 기반 공정 진행 및 BOM 데이터에 따른 자재 실시간 차감
STEP 04. 품질 검사 (QA): 의료기기 표준(DICOM) 기반 합격/불합격 판정 자동화
STEP 05. 출하 및 보고: 최종 공정 이력 데이터 확정 및 결과 보고서 생성

02. 역할 기반 권한 설계 (RBAC)

관리자
사원 권한 부여, 공정 기준 정보 설정, 생산 통계 대시보드 조회
작업자
실시간 공정 보고, 불량 발생 등록(코드), 개인 작업 지시서 확인
