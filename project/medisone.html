Medis-ONE

의료용 디스플레이 생산 공정 통합 관리 시스템 (MES)

Project Overview

의료용 디스플레이 제조 공정의 고도화된 품질 관리를 위해 **자재 입고부터 생산, 품질 검사, 최종 출하 및 사후 추적**까지 전 과정을 디지털화하고 통합 관리하는 MES(Manufacturing Execution System) 솔루션입니다.

🛠 Tech Stack

Spring Boot JPA / Hibernate MySQL React REST API

Key Module & Features

  • 자재 및 재고 관리 (Inventory)
    부품 입고 시 바코드 연동을 통한 실시간 재고 파악 및 불량 자재 선별
  • 실시간 생산 공정 모니터링 (Production)
    작업 지시서 하달 및 공정별 진행률(WIP) 실시간 가시화
  • 품질 관리 및 추적 (QA & Traceability)
    검사 데이터 자동 수집 및 시리얼 번호 기반의 출하 후 이력 추적(History Tracking) 기능
  • 데이터 시각화 (Dashboard)
    생산 효율성(OEE) 및 불량률 통계 대시보드 구현

💡 Implementation Focus

의료기기 품질경영시스템(ISO 13485) 기준을 참고하여 **데이터의 무결성**을 확보하는 데 집중했습니다. 특히 대규모 공정 데이터를 효율적으로 처리하기 위해 데이터베이스 정규화와 인덱싱 최적화를 진행했습니다.
